# 4-Bit ALU using Verilog HDL
## Overview

This project is a 4-bit Arithmetic Logic Unit (ALU) designed using Verilog HDL. It was developed as part of my RTL Design learning journey to strengthen my understanding of combinational logic design and Verilog coding.

The ALU performs multiple arithmetic and logical operations based on a 3-bit select input. The design was simulated and verified using Xilinx Vivado.

## Features

- 4-bit combinational ALU
- Addition
- Subtraction
- Bitwise AND
- Bitwise OR
- Bitwise XOR
- Bitwise NOT
- Left Shift
- Right Shift
- Carry output

## Inputs and Outputs

### Inputs

| Signal | Width | Description |
|--------|-------|-------------|
| A | 4-bit | First operand |
| B | 4-bit | Second operand |
| Sel | 3-bit | Operation selector |

### Outputs

| Signal | Width | Description |
|--------|-------|-------------|
| Result | 4-bit | ALU output |
| Carry | 1-bit | Carry generated during arithmetic operations |

## Supported Operations

| Sel | Operation |
|-----|-----------|
| 000 | A + B |
| 001 | A - B |
| 010 | A & B |
| 011 | A \| B |
| 100 | A ^ B |
| 101 | ~A |
| 110 | A << 1 |
| 111 | A >> 1 |
